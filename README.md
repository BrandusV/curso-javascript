# Curso de JavaScript - Anotações de Aula

Repositório com os códigos e anotações das aulas do curso de JavaScript. Cada pasta contém os exemplos práticos da aula correspondente, com comentários explicativos sobre os conceitos abordados.

| **Aula 4** | Funções Tradicionais e Arrow Functions | [`aula4/index.js`](aula4/index.js) |

| **Aula 9** | Estruturas Condicionais (`if/else`, `switch`, ternário) | [`aula9/index.js`](aula9/index.js) |

## Como usar

Cada arquivo pode ser executado individualmente com Node.js:

```bash
node aula1/index.js
```
