# Curso de JavaScript - Anotações de Aula

Repositório com os códigos e anotações das aulas do curso de JavaScript. Cada pasta contém os exemplos práticos da aula correspondente, com comentários explicativos sobre os conceitos abordados.

## Conteúdo das Aulas

| Aula | Tema | Arquivo |
|------|------|---------|
| **Aula 1** | Declaração de Variáveis (`var`, `let`, `const`) | [`aula1/index.js`](aula1/index.js) |
| **Aula 2/3** | Tipos de Dados (string, number, boolean, null, undefined) | [`aula2/index2.js`](aula2/index2.js) |
| **Aula 4** | Funções Tradicionais e Arrow Functions | [`aula4/index.js`](aula4/index.js) |
| **Aula 5** | Parâmetros de Funções | [`aula5/index.js`](aula5/index.js) |
| **Aula 6** | Funções de Ordem Superior e Funções Aninhadas | [`aula6/index.js`](aula6/index.js) |
| **Aula 7** | Retorno de Funções (`return`) | [`aula7/index.js`](aula7/index.js) |
| **Aula 8** | Escopo Global e Local | [`aula8/index.js`](aula8/index.js) |
| **Aula 9** | Estruturas Condicionais (`if/else`, `switch`, ternário) | [`aula9/index.js`](aula9/index.js) |

## Como usar

Cada arquivo pode ser executado individualmente com Node.js:

```bash
node aula1/index.js
```
